# 📊 Income, Expense & Savings Pattern Analysis Dashboard

## 📌 Project Overview

This project presents an **interactive Excel dashboard** that analyzes financial behavior across different demographic groups.
The analysis focuses on **City Tier, Occupation, and Number of Dependents** to understand patterns in **income, expenses, and savings potential**.

The dashboard provides insights into **cost of living, spending behavior, and savings patterns** using pivot tables, slicers, and dynamic charts.

---

## 🎯 Objectives

* Compare **income distribution across occupations**
* Analyze **expense patterns based on number of dependents**
* Evaluate **financial differences between city tiers**
* Identify **spending and savings behavior across demographic groups**

---

## 📂 Dataset Features

The dataset includes the following attributes:

* Income
* Age
* Occupation
* City Tier
* Dependents
* Rent
* Groceries
* Transport
* Healthcare
* Education
* Insurance
* Disposable Income
* Desired Savings
* Potential Savings

---

## 📊 Dashboard Sections

### 🏙 City Tier Analysis

Compares financial behavior across **Tier 1, Tier 2, and Tier 3 cities** to understand differences in income levels, living costs, and spending patterns.

---

### 💼 Occupation Analysis

Explores how different professions influence **income distribution and spending behavior**, highlighting financial differences between occupation groups.

---

### 👨‍👩‍👧 Dependents Analysis

Analyzes how the **number of dependents affects household expenses**, particularly in categories like groceries, education, and healthcare.

---

---

## 📊 Key Insights

* Higher income groups do not always translate into higher savings.
* Households with more dependents show significantly higher expenses.
* Tier 1 cities generally have higher income but also higher living costs.
* Certain occupations show higher potential savings opportunities.
